At level 15, go to your skill menu and select:

## YOUR SKILL BAR

```
[Basic]  ADVANCE        <- Still your Faith generator
[Core]   Blessed Hammer <- Damage
[1]      Falling Star   <- Jump to enemies + triggers Arbiter
[2]      Condemn        <- Groups enemies + Vulnerable
[3]      Defiance Aura  <- Keep active
[4]      Rally          <- Movement + emergency Faith
```

---

## HOW TO PLAY

### Before Level 22:
```
1. Falling Star INTO pack
2. Advance 2-3x (Faith)
3. Blessed Hammer 1-2x
4. Condemn if enemies spread out
5. Advance to refill
6. Repeat
```

### After Level 22 (Flash of the Blade):
```
1. Falling Star INTO pack (triggers Arbiter)
2. Advance (also triggers Arbiter now!)
3. Blessed Hammer 2-3x (they orbit you)
4. Condemn to group + Vulnerable
5. Falling Star to next pack
```

### After Level 34 (Exaltation):
**BIG DAMAGE SPIKE** - Game gets way easier

```
1. Falling Star -> Arbiter Form
2. Condemn -> Vulnerable + grouping
3. Blessed Hammer x3
4. Rally for speed
5. Repeat
```

## GEAR PRIORITY

1. **WEAPON DAMAGE** (always upgrade weapon)
2. +Blessed Hammer ranks
3. Cooldown Reduction
4. Attack Speed
5. Life/Armor

### Worth Keeping:
- Any weapon with +Blessed Hammer
- Gear with Cooldown Reduction
- Boots with Movement Speed
