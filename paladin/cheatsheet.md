# BLESSED HAMMER PALADIN - CHEAT SHEET

## THE BASICS

| What | Answer |
|------|--------|
| Class | Paladin |
| Build | Blessed Hammer (Hammerdin) |
| Tier | S-Tier |
| Oath | DISCIPLE (Leveling/Speed) or JUDICATOR (Push) |
| Key Passive | EXALTATION (Level 34) |

---

## SKILL BAR

### Leveling (1-34)
```
Basic:  ADVANCE (spam for Faith!)
Core:   Blessed Hammer
1:      Falling Star
2:      Condemn
3:      Defiance Aura
4:      Rally
```

### Endgame (35+)
```
Basic:  Rally
Core:   Blessed Hammer
1:      Falling Star
2:      Condemn
3:      Defiance Aura
4:      Fanaticism Aura
Ult:    Arbiter of Justice
```

---

## ROTATION

### Early Game (Before 34)
```
Advance x2 -> Hammer x1 -> Advance x2 -> Hammer x1
Rally when Faith empty
```

### Endgame
```
Falling Star -> Condemn -> Hammer x3 -> Evade spam
```

## PARAGON

**Board Order:** Starter -> CASTLE -> Relentless -> Shield Bearer -> Beacon

| Board | Glyph | Legendary Node |
|-------|-------|----------------|
| 1. Starter | Spirit | - |
| 2. **Castle** | **Exploit** | **Castle (1.8% Armor = Damage)** |
| 3. Relentless | Challenger | Dervish (CDR while moving) |
| 4. Shield Bearer | Sentinel | Shield Bearer (Block = Damage) |
| 5. Beacon | Resplendence | Resplendent (Aura resets) |

**Priority:** Rush Castle board first - it converts your Armor into damage.

## TROUBLESHOOTING

| Problem | Fix |
|---------|-----|

| Low damage | Upgrade weapon |
